# M_Skills

## Scope

个人常用 Skills 收集仓库，记录和沉淀日常使用的工作流、脚本、配置模板。

## Mission

- 收集并整理开发、运维、AI、生活等场景下的实用技能
- 每条 Skill 需包含：场景描述、前置条件、操作步骤、验证方法
- 脚本放在 `scripts/`，可复用的配置模板放在 `templates/`

## 目录

| 分类 | 目录 | 说明 |
|---|---|---|
| 开发 | `skills/dev/` | 编码、调试、CI/CD、工具链 |
| 运维 | `skills/ops/` | 部署、服务器管理、网络 |
| AI | `skills/ai/` | LLM、Prompt 工程、模型部署 |
| 生活 | `skills/life/` | 日常技巧、工具使用 |

## Skill 模板

```markdown
---
name: skill-name
tags: [tag1, tag2]
created: YYYY-MM-DD
---

# Skill 标题

## 场景
什么情况下使用

## 前置条件
需要什么环境/工具/权限

## 步骤
1. xxx
2. xxx

## 验证
如何确认完成

## 参考
相关链接
```
