## Local Notes

- Bash 脚本保持 `set -euo pipefail`；`install-user-skills.py` 使用标准库，需支持 Windows / Linux / macOS。
- 新增或修改 Bash 脚本后至少运行 `bash -n scripts/*.sh`；修改 `install-user-skills.py` 后运行 `python3 -m py_compile scripts/install-user-skills.py`。
- 安装目标或输出格式变化时，同步更新 `docs/installation.md` 和 `docs/repo-init.md`。
- `install-user-skills.py` 启动时要求 Bitwarden CLI `bw` 已安装且已登录；随后从 `skills/<skill-name>/SKILL.md` 安装本地用户级 skills；末尾会安装/检查 GitHub CLI `gh`，尝试用 `gh skill install --from-local` 安装本地用户级 skills，随后安装 [playwright-cli](https://github.com/microsoft/playwright-cli) 与 skill，并从 ima agent-interface 安装 `ima-skill`、交互写入 `~/.config/ima/` 凭证。
